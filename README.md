# Uber Supply Demand Gap Analysis

## Project Overview

This project analyzes Uber ride request data to identify the supply-demand gap using Google Sheets, PostgreSQL, and Python.

## Tools Used

- Google Sheets
- PostgreSQL
- Python
- Pandas
- NumPy
- Matplotlib

## Dataset

Uber Request Data.csv

## Project Workflow

1. Data Cleaning
2. Dashboard Creation
3. SQL Analysis
4. Python EDA
5. Business Insights

## Key Findings

- Most ride requests were completed successfully.
- Airport has more "No Cars Available" requests.
- Morning and Evening are peak demand periods.
- Driver cancellations contribute to supply-demand gaps.

## Recommendations

- Increase driver availability during peak hours.
- Improve ride allocation.
- Reduce cancellations through better scheduling.
